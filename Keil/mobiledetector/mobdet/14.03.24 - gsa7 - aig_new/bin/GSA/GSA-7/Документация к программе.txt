:: AIG2

Программа для настройки АИГ2.

Системные требования:
 - .NET Framework 4 (full profile)


:: Особенности работы

После подключения происходит однократное чтение значений
параметров из устройства. В дальнейшем происходит только
запись установленных значений. При изменении значений в
устройстве во время работы программы, они будут перезаписаны.

Для изменения настроек можно использовать движки, перемещая
ползунок мышью или клавишами стрелок (после выделения ползунка).

Для ввода точного значения нужно щелкнуть на надписи со
значением.

Кнопки "+" и "-" рядом с ползунком позволяют изменять
значение с фиксированным шагом. Шаг можно увеличить x10,
зажав клавишу "Control", или уменьшить, зажав "Shift".

Изменение масштаба отображения производится мышью.
Левой кнопкой мыши можно выделить участок для увеличения.
Правая кнопка мыши восстанавливает масштаб по умолчанию.
Колесо мышки меняет масштаб на фиксированное значение.
Прокрутка координат производится средней кнопкой мыши.


:: Отладочная консоль

poller( 0|1 ) - включение постоянного опроса
interval( sec ) - изменение периода опроса, секунды, вещественное
start() - начать запись
stop() - завершить запись
connect( com ) - подключение
disconnect() - отключение
r( start[,count] ) - чтение регистра или нескольких регистров
w( address,value ) - запись регистра
reread() - перечитать значения из устройства
update() - цикл обновления (чтение и запись)
print( a ) - печать в консоль
sleep( sec ) - пауза выполнения сценария
